# Support

Thanks for using GLM Code.

## Questions and usage help

Use GitHub Discussions for questions, setup help, troubleshooting, and general
community conversation:

- https://github.com/qazz92/glm-code/discussions

## Bugs and feature requests

Use GitHub Issues when you have a reproducible bug report or a concrete feature
request:

- https://github.com/qazz92/glm-code/issues/new/choose

Before opening a new issue, please search existing issues and include enough
information for maintainers to reproduce or evaluate the request.

## Security reports

Do not open public issues for suspected vulnerabilities. Follow the instructions
in [SECURITY.md](./SECURITY.md).
